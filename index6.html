CSS (Cascading Style Sheets)

What is CSS, How does it Relate to HTML...

CSS (Cascading Style Sheets) is used to control the presentation of HTML (HyperText Markup Language) content. While HTML structures and defines the content of a web page using elements like headings, paragraphs, and images, CSS applies styles to these elements, such as colors, fonts, spacing, and layout.

By separating (content) HTML from design (CSS), developers can maintain and update the visual appearance of a website more efficiently, ensuring a consistent look and feel across multiple pages while keeping the HTML code clean and focused on structure.


How to apply it to HTML....

We can style using <div> ,<span>, in-line tags (inside the opening tags) as we have seen before. css property can be applied for specific needs this way. But, that can be redundant for longer html documents with lots of lines of code.

The preferred way to do it externally. To write css code on a seperate style.css file.
and then the html documents can get those styling properties with a line of code which tells that respective html document that it wants to be related to that css styling file.

Multiple html documents can achieve consistent styling by using the <link> element to connect them to a single CSS file.

<link rel = "stylesheet" href = "style.css">

Obviously include this within the head tag and not anywhere else. Because head tags serve the purpose of having meta information, scripts for javascript, styling for css, titles and other related information.
