CSS (Cascading Style Sheets)

What is CSS, How does it Relate to HTML...

CSS (Cascading Style Sheets) is used to control the presentation of HTML (HyperText Markup Language) content. While HTML structures and defines the content of a web page using elements like headings, paragraphs, and images, CSS applies styles to these elements, such as colors, fonts, spacing, and layout.

Topics to be Covered ...

Classes & IDs, Font Size & Layout, Positioning, CSS Box Model (Margin, Border, Padding, Content), CSS Units (px, em, rem) and more.

Use CSS styling with <style> tags (Internal) in the head section. Inline styling with <div> and <span> tags can be redundant for long documents. The preferred way is to use an external style.css file. Link it using <link rel="stylesheet" href="style.css"> within the head tag.
