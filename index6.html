CSS (Cascading Style Sheets)

CSS (Cascading Style Sheets) is used to control the presentation of HTML (HyperText Markup Language) content. While HTML structures and defines the content of a web page using elements like headings, paragraphs, and images, CSS applies styles to these elements, such as colors, fonts, spacing, and layout.

By separating content (HTML) from design (CSS), developers can maintain and update the visual appearance of a website more efficiently, ensuring a consistent look and feel across multiple pages while keeping the HTML code clean and focused on structure.
