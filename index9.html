Web Layout and Float

Web Layout

Even though it is up to personal preference, the common website layout/format is as follows:

Header:

The header is typically the top section of a webpage.
It often contains the website's logo, title, and navigation links.
Think of it as the area that introduces the site and gives users their first impression.

Navigation (Nav):

This section provides links to the main parts of the website.
It helps users find their way around the site easily.
The nav element usually contains a list of links organized in a menu format.

Main Content:

This is the primary area where the main content of the webpage is displayed.
It can include several subsections:

Section: A section is a thematic grouping of content, typically with a heading. It's like a chapter in a book.

Article: An article is a self-contained piece of content that can stand alone. It could be a blog post, news article, or any independent content item.

Aside: The aside element contains content that is tangentially related to the main content. It’s often used for sidebars, pull quotes, or additional information.

Footer:

The footer is located at the bottom of the webpage.
It often contains copyright information, links to privacy policies, contact details, and sometimes additional navigation links.
Think of it as the closing section that wraps up the webpage.

Visual Layout Example:

Developers frequently use the <div> element to organize their code into different sections. The <div> element can also be used in CSS for positioning these sections. However, using too many <div> elements can make the code difficult to read because <div> itself doesn’t provide much information about the content it contains.

This is why semantic HTML tags are preferred for web layout. Semantic tags like <header>, <nav>, <main>, <section>, <article>, <aside>, and <footer> give more meaning to the structure of the webpage. They describe the role of the content they contain, making the code more readable and understandable.

What is Float?

In short, the float property in CSS is used to position elements horizontally on a webpage, as if they are "floating" to the left or right within their containing element. It allows text and inline elements to wrap around the floated element, creating various layout possibilities.

The article, section, and aside elements on this webpage of which each have implemented float: float: left;

Why We Use Float:

To position elements side by side, such as images and text.
To create multi-column layouts.
To wrap text around images or other elements.
